## Project Description
  This project makes a cafe kiosk system using Java's Object-Oriented Programming(OOP). 
We use "Inheritance" to organize many menus like Coffee and Dessert easily. 
Also, we use "Polymorphism" to handle different payments like Credit Cards and Cash, making the code flexible. 
Users can see the menu, put items in a cart, and pay through the console screen. 
We also plan to add an "Administrator Mode" to check total sales and manage stock, just like a real cafe kiosk.
